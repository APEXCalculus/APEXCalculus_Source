{$\ds f(x,y) = \frac{1}{x^2+y^2+1}$, $P = (1,1)$.
}
{$\nabla f = \la \frac{-2x}{(x^2+y^2+1)^2}, \frac{-2y}{(x^2+y^2+1)^2}\ra$
\begin{enumerate}
	\item  $\nabla f(1,1) = \la -2/9,-2/9\ra$.
	\item		$\norm{\nabla f(1,1)} = \norm{\la -2/9,-2/9\ra}=2\sqrt{2}/9$
	\item		$\la 2/9,2/9\ra$
	\item		$\la 1/\sqrt{2},-1/\sqrt{2}\ra$
\end{enumerate}
